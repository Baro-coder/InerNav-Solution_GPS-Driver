NMEA CRC calculate function.

Processing the binary XOR operation between all characters in the sentence (between NMEA_START_CHAR and NMEA_CONTROL_CHAR)

Parameters
msgNMEA message buffer handler
Returns
On SUCCESS : uint16_t The checksum validation value. On ERROR : 0x00 value
